# اللجان والـ RACI

## 1. اللجان الأساسية

### Clinical Safety Board

الجهة العليا للسلامة السريرية داخل النظام.

المسؤوليات:

- اعتماد السياسات السريرية عالية الخطورة
- مراجعة المخاطر الحرجة
- مراجعة الحوادث ذات الأثر الكبير
- اعتماد النشر للمحتوى عالي الخطورة

التشكيل المقترح:

- المدير الطبي
- مدير التمريض
- رئيس الصيدلة
- مدير الجودة وسلامة المرضى
- ممثل المختبر
- ممثل الأشعة
- قائد المنتج/التقنية

### Medication Safety Committee

الجهة المالكة لسلامة الدواء.

المسؤوليات:

- مراجعة التنبيهات الدوائية
- مراجعة تجاوزات التنبيهات `override`
- اعتماد قواعد الأدوية
- مراجعة أخطاء الصرف والإعطاء

### Clinical Content Committee

الجهة المالكة للمحتوى السريري المعياري.

المسؤوليات:

- اعتماد `order sets`
- اعتماد `clinical pathways`
- اعتماد القوالب السريرية
- تحديد المراجعة الدورية للمحتوى

### Nursing Informatics Council

الجهة المالكة لسير التمريض داخل النظام.

المسؤوليات:

- handover standards
- nursing documentation standards
- eMAR workflow
- تعليمات التنفيذ والرعاية اليومية

### Diagnostic Governance Board

الجهة المالكة للقيم الحرجة ومسارات المختبر والأشعة.

المسؤوليات:

- critical values policy
- escalation policy
- acknowledgment workflow
- turnaround time review

## 2. RACI مختصر للمجالات الأساسية

| المجال | Responsible | Accountable | Consulted | Informed |
| --- | --- | --- | --- | --- |
| سياسات السلامة السريرية | Clinical Safety Board | المدير الطبي | الجودة + التقنية | الإدارة التنفيذية |
| قواعد سلامة الدواء | Medication Safety Committee | رئيس الصيدلة | الأطباء + التمريض + التقنية | الجودة |
| `Order Sets` | Clinical Content Committee | رئيس القسم السريري المعني | الصيدلة + المختبر + الأشعة + التقنية | المدير الطبي |
| `Clinical Pathways` | Clinical Content Committee | المدير الطبي أو رئيس الخدمة | التمريض + الجودة + التقنية | الإدارة التنفيذية |
| `CDSS` rules | Clinical Safety Board + Medication Safety Committee | المدير الطبي | التقنية + الجودة | الأقسام المعنية |
| القيم الحرجة | Diagnostic Governance Board | مدير المختبر / مدير الأشعة | التمريض + الأطباء + التقنية | الجودة |
| handover workflow | Nursing Informatics Council | مدير التمريض | الجودة + التقنية | المدير الطبي |
| incident / near miss / CAPA | الجودة وسلامة المرضى | المدير الطبي | جميع اللجان | الإدارة التنفيذية |
| release approval للمحتوى عالي الخطورة | Clinical Safety Board | المدير الطبي | قائد التقنية + الجودة | الأطراف المعنية |

## 3. وتيرة الاجتماعات

| اللجنة | الوتيرة |
| --- | --- |
| Clinical Safety Board | شهريًا، واجتماع طارئ عند الحاجة |
| Medication Safety Committee | كل أسبوعين |
| Clinical Content Committee | أسبوعيًا |
| Nursing Informatics Council | شهريًا |
| Diagnostic Governance Board | شهريًا |

## 4. مخرجات كل اجتماع

كل اجتماع يجب أن ينتج:

- قرارات معتمدة أو مرفوضة
- عناصر إجراء `Action items`
- مالك لكل عنصر
- تاريخ استحقاق
- مستوى خطورة إن وجد
- قرار نشر أو إرجاع للمراجعة

## 5. قواعد القرار

- لا يعتمد أي محتوى سريري عالي الخطورة بدون مراجع سريري أساسي ومراجع سلامة.
- أي اعتراض متعلق بسلامة المريض يوقف النشر لحين الحسم.
- أي نشر طارئ يجب أن يسجل كـ `Emergency Clinical Change` مع مراجعة لاحقة خلال 5 أيام عمل.
